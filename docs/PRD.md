# 云相册 PRD

## 1. 背景

本项目是一个给个人和家人使用的私有云相册。产品以移动端体验优先，核心目标是安全保存、浏览、上传和管理家庭照片与视频。

技术前提：

- 前端使用 Next.js。
- 后端业务操作优先使用 Server Action。
- 文件存储使用腾讯云 COS。
- 元数据存储使用 PostgreSQL。

## 2. 产品目标

第一版目标：

- 支持手机号加密码登录。
- 不开放自由注册，只允许全局管理员发出注册邀请链接。
- 支持用户创建和加入多个空间。
- 支持一级文件夹管理。
- 支持图片和视频上传、浏览、预览、删除和恢复。
- 支持多文件上传、视频分片上传、失败重试和续传。
- 使用私有 COS 存储，保护家庭照片和视频隐私。

第一版不做：

- 自由注册。
- 短信验证码登录。
- 多级文件夹。
- 搜索。
- 自动清理回收站。
- 复杂图片压缩、视频转码、视频封面生成。

## 3. 用户与角色

### 3.1 全局管理员

全局管理员是应用级别的管理者，第一版只有开发者本人。

全局管理员能力：

- 创建注册邀请链接。
- 撤销注册邀请链接。
- 查看和管理所有账号。
- 禁用账号。
- 查看普通界面不可见的永久删除记录。
- 必要时进行后台恢复。

### 3.2 普通用户

普通用户通过注册邀请链接创建账号。

普通用户能力：

- 使用手机号和密码登录。
- 自由创建空间。
- 加入已有空间。
- 在所属空间内管理文件夹和媒体。
- 退出空间。

### 3.3 空间成员

空间内第一版不区分 owner、admin、member 权限。凡是加入空间的用户，在该空间内拥有同等操作权限。

产品展示名称统一使用“成员”。

空间成员能力：

- 浏览空间内文件夹和媒体。
- 创建文件夹。
- 上传图片和视频。
- 删除媒体。
- 删除整个文件夹。
- 从回收站恢复内容。
- 将已注册用户加入当前空间。
- 创建者可以移除当前空间的其他成员。
- 退出当前空间。

## 4. 注册与登录

### 4.1 注册邀请

第一版不开放自由注册。账号注册只能通过全局管理员生成的邀请链接完成。

邀请规则：

- 邀请链接绑定一个手机号。
- 邀请链接页面显示该手机号，用户不能修改。
- 用户通过邀请链接填写昵称、设置密码并完成注册。
- 注册完成后不自动加入任何空间。
- 已注册手机号不能再次创建邀请。
- 同一个手机号不能同时存在多个未处理邀请。
- 邀请没有有效期。
- 邀请可以由全局管理员撤销。
- 邀请撤销后，可以为同一手机号重新生成邀请。

### 4.2 登录

登录方式：

- 手机号。
- 密码。

登录成功后：

- 登录后默认进入空间列表。
- 如果没有任何空间，进入空状态，引导用户创建空间。

## 5. 空间

空间是相册数据的隔离单位。文件夹、媒体、回收站记录都必须归属于某个空间。

空间规则：

- 用户可以自由创建空间。
- 一个用户可以加入多个空间。
- 空间成员只能添加已经注册成功的用户。
- 空间详情页提供成员管理入口。
- 成员管理页展示当前空间成员，邀请时只填写手机号。
- 成员管理页中，只有空间创建者可以移除其他成员。
- 空间创建者不能被移除。
- 成员管理页可以退出当前空间。
- 空间创建者不能退出空间。
- 空间内第一版全员同权。
- 用户可以退出空间。
- 如果最后一个成员退出，空间变为无成员空间，普通用户不可见，全局管理员仍可管理。

## 6. 文件夹

第一版只支持一级文件夹。

文件夹能力：

- 创建文件夹。
- 查看文件夹卡片视图。
- 查看文件夹列表视图。
- 进入文件夹查看图片和视频。
- 删除整个文件夹。
- 从回收站恢复文件夹。

文件夹删除规则：

- 删除文件夹时，仅做逻辑删除。
- 文件夹内未删除媒体同步进入回收站。
- 本次删除的文件夹和媒体使用同一个删除批次标识。
- 恢复文件夹时，一起恢复本次随文件夹删除的媒体。
- COS 文件不删除。

## 7. 媒体

媒体包括图片和视频。

媒体能力：

- 多文件上传。
- 图片上传。
- 视频上传。
- 图片网格浏览。
- 视频网格浏览。
- 图片沉浸式预览。
- 视频播放预览。
- 删除媒体。
- 长按媒体进入多选，支持拖动快速选择、批量删除。
- 长按媒体进入多选，支持拖动快速选择、批量下载。
- 媒体预览更多菜单支持下载和删除。
- 从回收站恢复媒体。
- 永久删除媒体。

媒体排序：

- 默认按拍摄时间排序。
- 图片优先读取 EXIF 拍摄时间。
- 视频优先读取可用媒体拍摄时间。
- 读取失败时，使用上传时间。

媒体元数据：

- 文件名。
- 文件类型。
- MIME 类型。
- 文件大小。
- COS object key。
- 上传人。
- 所属空间。
- 所属文件夹。
- 拍摄时间。
- 上传时间。
- 图片宽高。
- 视频宽高。
- 视频时长。
- 上传状态。
- 删除状态。

## 8. 上传

上传使用浏览器直传腾讯云 COS，Server Action 负责业务状态和授权。

上传流程：

1. 用户选择目标空间和文件夹。
2. 用户选择一个或多个文件。
3. Server Action 创建上传记录。
4. Server Action 返回 COS 上传所需的临时授权或签名信息。
5. 浏览器将文件直传至 COS。
6. 大文件和视频使用分片上传。
7. 上传失败时支持重试和续传。
8. 上传完成后，前端调用 Server Action 确认上传完成。
9. Server Action 更新媒体状态为可用。

上传限制：

- 第一版不主动限制单文件大小。
- 实际限制以 COS、浏览器和部署环境能力为准。

## 9. 回收站与删除

删除分为普通删除和永久删除。

普通删除：

- 内容进入回收站。
- 删除前必须二次确认。
- 记录删除人。
- 记录删除时间。
- COS 文件不删除。
- 可从回收站恢复。

永久删除：

- 仍然只做逻辑删除。
- 普通空间界面不可见。
- 普通用户不可恢复。
- COS 文件不删除。
- 全局管理员后台仍可查看记录，并保留紧急恢复能力。

回收站规则：

- 不自动清理。
- 回收站保持和普通相册一致的一级文件夹结构。
- 回收站首页按文件夹展示，不直接混排所有已删除媒体。
- 进入回收站内的文件夹后，展示该文件夹下已删除的图片和视频。
- 单独删除的媒体仍归属原文件夹，在回收站中按原文件夹分组展示。
- 回收站列表中的删除信息统一展示为：删除人、删除时间、数量。
- 文件夹恢复时，需要恢复同一删除批次下随文件夹删除的媒体。

## 10. 隐私与安全

隐私目标：

- 家庭照片和视频不能公开暴露。
- COS bucket 应使用私有读写。
- 图片和视频访问通过服务端生成短期签名 URL。
- 所有业务查询必须按空间隔离。
- 用户只能访问自己所属空间的数据。
- Server Action 必须校验当前用户是否属于目标空间。

安全规则：

- 数据库不保存明文密码。
- 注册邀请校验使用 token 哈希；为支持管理员再次复制待注册邀请链接，待注册邀请同时保存原始 token，且只在全局管理后台返回给全局管理员。
- COS 访问密钥不得暴露到客户端。
- 客户端只能获得完成指定上传或读取所需的最小授权。

## 11. 页面与功能

### 11.1 登录页

- 手机号输入。
- 密码输入。
- 登录按钮。
- 错误提示。

### 11.2 邀请注册页

- 通过邀请 token 访问。
- 展示被邀请手机号。
- 填写昵称。
- 设置密码。
- 确认密码。
- 完成注册。

### 11.3 空间入口

- 用户有空间时进入默认空间。
- 用户有多个空间时支持切换空间。
- 用户无空间时展示创建空间入口。

### 11.4 文件夹首页

- 展示当前空间内一级文件夹。
- 支持卡片视图。
- 支持列表视图。
- 支持进入成员管理页。
- 支持创建文件夹。
- 支持上传入口。
- 支持进入文件夹。

第一版不做搜索。

### 11.4.1 成员管理页

- 展示当前空间名称。
- 展示当前空间成员列表。
- 成员列表展示昵称、手机号、成员身份和加入时间。
- 支持邀请已注册用户加入当前空间。
- 邀请时只填写手机号。
- 支持创建者移除其他成员。
- 创建者不能被移除。
- 支持当前用户退出空间。
- 空间创建者不展示退出空间入口，服务端也拒绝创建者退出。
- 移除成员和退出空间前必须二次确认。

### 11.5 文件夹详情页

- 展示当前文件夹内图片和视频。
- 图片和视频混排。
- 支持按类型筛选。
- 默认按拍摄时间排序。
- 支持上传媒体。
- 支持删除媒体。
- 支持长按进入多选。
- 多选状态支持拖动范围快速选择：拖动开始项到手指当前位置之间的媒体组成当前选择范围；开始项未选中时选中该范围，开始项已选中时取消选中该范围；手指往回拖时范围同步收缩，不能影响普通纵向滑动。
- 多选后支持批量下载。
- 多选后支持批量删除。
- 删除媒体前必须二次确认。
- 支持进入预览。

### 11.6 媒体预览页

- 图片沉浸式预览。
- 视频使用原生播放器预览，未播放或暂停状态保留浏览器原生长按保存能力。
- 支持左右切换。
- 左右切换时暂停正在播放的视频。
- 页面切到后台时暂停正在播放的视频。
- 视频预览的原生控件不能被底部缩略图栏遮挡。
- 视频播放中隐藏预览顶部栏和底部缩略图栏。
- 视频播放中支持长按画面临时 2 倍速，松手恢复正常倍速。
- 支持返回。
- 支持更多菜单。
- 更多菜单支持下载。
- 更多菜单支持删除。
- 删除媒体前必须二次确认。
- 控制信息保持克制。

### 11.7 回收站

- 按文件夹展示普通删除内容。
- 文件夹列表展示删除人、删除时间和内容数量。
- 进入文件夹后查看该文件夹下已删除媒体。
- 支持恢复。
- 支持永久删除。

### 11.8 全局管理后台

第一版只面向全局管理员。

能力：

- 创建注册邀请。
- 撤销注册邀请。
- 查看账号。
- 禁用账号。
- 查看永久删除记录。
- 必要时进行后台恢复。

界面规则：

- 管理后台按管理对象分区展示，不把邀请、账号、删除记录混在同一个列表中。
- 邀请管理页包含创建邀请表单和邀请记录列表。
- 邀请列表不展示已撤销邀请。
- 待注册邀请需要展示邀请链接，便于管理员再次复制。
- 账号管理页支持禁用账号；禁用后账号不能登录，已有会话失效，历史上传、删除和空间记录保留；管理员不能禁用当前登录账号。

## 13. 验收标准

第一版完成时应满足：

- 只有全局管理员可以创建注册邀请。
- 用户可以通过邀请链接设置密码注册。
- 用户可以使用手机号和密码登录。
- 用户可以创建空间。
- 用户可以加入多个空间。
- 用户默认进入空间列表，不自动打开某个空间。
- 用户只能看到所属空间的数据。
- 空间成员可以通过手机号邀请已注册用户加入当前空间。
- 空间创建者可以在成员管理页移除其他成员。
- 空间创建者不能被移除。
- 空间成员可以在成员管理页退出当前空间。
- 用户可以创建一级文件夹。
- 用户可以上传多张图片。
- 用户可以上传视频。
- 大文件和视频上传支持分片、重试和续传。
- 媒体默认按拍摄时间排序，读取失败时使用上传时间。
- 用户可以删除媒体和文件夹。
- 删除媒体前有二次确认。
- 用户可以长按媒体多选，在选择状态拖动快速选择，并对所选媒体删除或下载。
- 用户可以在媒体预览更多菜单中删除当前媒体。
- 删除内容进入回收站。
- 删除文件夹时，文件夹内媒体一起进入回收站。
- 恢复文件夹时，随文件夹删除的媒体一起恢复。
- 永久删除后普通用户不可见，COS 文件仍保留。
- 第一版不出现搜索入口或搜索入口不可用状态。
- 私有 COS 文件不能通过长期公开 URL 访问。
